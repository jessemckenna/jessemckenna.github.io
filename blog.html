CS290 Web Development Project Website

2018-3-4

This is a simple sample blog for the CS290 final project. I researched many Content Management Systems for handling blog entries, but ultimately decided that a simple page written in HTML/CSS would be better for the final project, to minimize automatically created content and make it easy to grade.
