Traveling-Salesman Solver

Python

A 2-Opt approximation solver for the Traveling Salesman Problem.

This was my final project for Algorithms, completed as a team of three. The program had to handle extremely large sets of city coordinates under a time constraint and produce an accurate result, so this project was a fantastic way to put efficient coding into practice and see the incremental benefits in real time.

The program begins by finding a greedy starting path, then repeatedly selects two path nodes and checks whether the path length would be improved by swapping them. This swapping process continues until a certain amount of node pairs have been checked in a row with no potential improvement found, or time runs out.
